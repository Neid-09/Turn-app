## Ejecución /frontend

1. Clonar el repositorio

   ```bash
   git clone https://github.com/tu_usuario/Turn-app.git
   ```

2. Navegar al directorio del proyecto

   ```bash
   cd Turn-app/fronted/front
   ```

3. Instalar las dependencias

   ```bash
   npm install
   ```

4. Iniciar la aplicación

   ```bash
   npm run dev
   ```
